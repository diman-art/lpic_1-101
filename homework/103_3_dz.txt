Перед выполнением заданий важно самостоятельно повторить шаги, выполняемые на уроке.

1) Найти все файлы больше 100Мб в вашей ОС.
find / -size +100M
2) Найти все файлы больше 1Мб в домашней папке пользователя и перенести в корневую папку, предварительно сжав.
find /home/$USER -size 1M 2&>dev/null| xargs bzip2
mv *.bz /root
3) Найти в домашней папке все файлы, начианющиеся с букв "a" по "z" и скопировать их в папку temp в домашнем "каталоге".
mkdir -p $HOME/temp && find $HOME -name [a-z] -exec cp -r {} $HOME/temp \;
4) Сделать резервную копию загрузочного сектора диска.  
fdisk -l
sudo dd if=/dev/sda of=/tmp/mbr.img_backup bs=446 count= 1
5) Удалить все ненужные временные файлы.
find /tmp \! -atime -365 -exec rm -r {}\;    # Удалит все файлы ,к которым не обращались в течении года
6) Удалить все созданные в течении выполнения ДЗ каталоги.
Чтобы не удалить лишнего, перед началом работы включим снифер изменений в домашней папке:
apt install inotify-tools -y && inotifywait -r -m -e create /home | while read line; do echo "Created: $line"; done
Затем на основе полученных данных удалим созданные папки
