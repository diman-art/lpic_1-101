Перед выполнением заданий важно самостоятельно повторить шаги, выполняемые на уроке.

1) Добавить в репозитории информацию для работы с пакетом java.
sudo add-apt-repository ppa:linuxuprising/java;sudo apt update -y
2) Установить java со всеми зависимостями.
sudo apt install oracle-java12-installer
3) Удалить java.
apt-get autoremove oracle-java12-installer
4) Установить webmin.
apt install webmin
5) Обновить все пакеты ПО. 
apt-get update -y; apt-get upgrade
